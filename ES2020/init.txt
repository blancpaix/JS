splice 함수 좀 알아봐야 하고
sort 라는 함수가 이게 그냥 정렬을 해주는게 아니고 사전순으로 정렬을한다는데
첫번째 숫자만 따와서 정렬을 하느넉
이거 보통 윈도우 파일 정렬하는 형식으로 정렬을 함
ex) 30 50 40 7 => 30 40 50 7 이렇게 나옴
그래서 주의해서 쓰세요
그러면 숫자 크기순으로 정렬을 하는거면
sort 함수안에 arg 추가를 함
sort((p, c) => { return p-c })    // prev, current
리턴하는 값이 0 보다 크면 자리를 바꿈 그냥 순서대로 2개씩 묶어서 처리를함..
sort((p,c) => p-c)    // 오름차순
sort((p,c) => c-p)    // 내림차순 정렬     리턴값이 0보다 큰지 작은지가 중요
